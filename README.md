## 🏋️ Customer Fitness Product Analysis

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on customer fitness product data to understand customer behavior, product usage patterns, income distribution, and fitness preferences.

The goal is to identify key customer segments and generate business insights that can help improve marketing strategies and product recommendations.

---

## 🎯 Objectives

* Analyze customer demographics
* Understand product usage behavior
* Identify factors influencing fitness product purchases
* Discover trends in income, age, fitness levels, and usage patterns
* Generate actionable business insights

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📊 Dataset Features

The dataset contains information such as:

* Product Purchased
* Age
* Gender
* Education
* Marital Status
* Usage Frequency
* Fitness Rating
* Income
* Miles Run Per Week

* Bivariate Analysis

## 📈 Key Insights

* Identified the most popular fitness products.
* Analyzed customer age distribution.
* Studied income patterns among customers.
* Examined product usage frequency.

* Explored relationships between income, fitness level, and product usage.
